# STANAG 4774 / 4778 Compliance

This module implements NATO STANAG 4774 (Confidentiality Metadata) and 4778 (Security Metadata Binding).

## Classification Markings
- NATO UNCLASSIFIED, RESTRICTED, CONFIDENTIAL, SECRET, COSMIC TOP SECRET
- TLP:CLEAR, GREEN, AMBER, AMBER+STRICT, RED

## Binding Mechanism
Security labels bound to payloads via:
- Cryptographic signing (Ed25519)
- Hash chaining in audit log
- mTLS for transport
